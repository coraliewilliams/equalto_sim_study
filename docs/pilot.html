Meta-analysis models in glmmTMB: pilot simulation study

Coralie Williams

2025-10-29

This vignette describes the pilot simulation study results assessing the equalto covariance implementation in glmmTMB (Brooks et al., 2019). The simulation study aims to compare the new implementation with the metafor R package (Viechtbauer, 2010) for validation and proof-of-concept.

For demonstration purposes, this pilot study assesses feasibility by simulating a single dataset, examining one condition per parameter, and fitting all models planned for the full simulation study.

We will investigate four effect size measures:

  • Standardised Mean Difference (SMD)

  • Log Response Ratio (lnRR)

  • Odd ratio effect sizes (OR)

  • Poisson meta-analysis to model incidence rate ratios (IRR)

Notation

  • \(y_i\): observed effect size for study \(i\).
  • \(k\): number of studies.
  • \(\mu\): overall mean effect on the analysis scale (SMD, lnRR, log-OR, log-IRR).
  • \(\tau^2\): between-study (random-effects) variance.
  • \(\mathbf{V}\): block-diagonal sampling-error variance–covariance matrix.
  • \(\sigma_T, \sigma_C\): within-group standard deviations for treatment and control (SMD/lnRR).
  • \(\mu_C\): control-group mean on the original scale (SMD/lnRR).
  • \(n_T, n_C\): sample sizes in treatment and control groups (SMD/lnRR/OR).
  • \(t_T, t_C\): exposure times (person-time) for treatment and control groups (IRR).


Model specification

For all effect size measures we will assess the random effect meta-analysis model:

\[y_{i} = \mu + u_{i} + e_{i} \tag{1}\] \[u \sim N(0,\tau^2)\] \[e \sim N(0,V)\]

Additionally, for non continuous effect size measures (OR and IRR), we will assess Binomial-Normal model and Poisson-Normal models.

Code

Fit random effects meta-analysis with metafor and glmmTMB:

Code 
# fit metafor model
m.smd.rma <- rma(yi, vi, data=dat.smd, control=list(REMLf=FALSE))

# fit glmmTMB model
dat.smd$g <- 1
dat.smd$id <- as.factor(1:nrow(dat.smd))
V <- diag(dat.smd$vi, nrow = nrow(dat.smd))
m.smd.tmb <- glmmTMB(yi ~ 1 + equalto(0 + id|g,V), ## the map variable "id" must be a factor
                    REML = TRUE,
                    data=dat.smd)

(3) Case study: Phylogenetic meta-analysis with dependent effect sizes

Here I demonstrate how to fit a phylogenetic meta-analysis with dependent effect sizes with glmmTMB using a simulated dataset.

\[y_{ijk} = \mu + u_{ij} + s_i + n_k + p_k + e_{ij}\] \[u \sim N(0,\sigma^2_{u}I_u)\] \[s \sim N(0, \sigma^2_sI_s)\] \[n \sim N(0, \sigma^2_nI_n)\] \[p \sim N(0, \sigma^2_p\textbf{A})\] \[e \sim N(0,V)\]

where \(\sigma^2_s\) the study level variance component, \(\sigma^2_u\) the estimate level variance component, \(\sigma^2_n\) is the variance of the species level random effect, \(\sigma^2_p\) is the variance of phylogenetic random effect, \(\textbf{V*}\) the block diagonal variance-covariance matrix of the sampling error with effect sizes from the same study, and \(\textbf{A}\) is the phylogenetic correlation matrix based on a Brownian model of evolution.

Code 
# Load some phylo dataset and matrix
load(here("data", "phylo_simulated_data.RData"))

Fit model using metafor:

Code 
phylo_rma_fit <- rma.mv(yi,
                   V,
                   random = list(~ 1| id, ~ 1 | study, ~ 1 | species, ~ 1 | species.phylo),
                   R = list(species.phylo=P),
                   test = "t",
                   dfs = "contain",
                   data = dat)

Fit model using glmmTMB:

Code 
phylo_tmb_fit <- glmmTMB(yi ~ 1 + equalto(0 + id|g,V) + (1|study) + (1|species) + propto(0 + species|g,P),
                       data=dat,
                       REML=T)

Compare output of metafor and glmmTMB:

Code

                  data.frame(f = "rma.mv", 
                        logLik = logLik(phylo_rma_fit),
                        df = attr(logLik(phylo_rma_fit), "df"), 
                        est = as.numeric(b), 
                        se = se,
